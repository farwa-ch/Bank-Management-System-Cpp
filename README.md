## üß© System Design
Each bank account is represented as an object of a class.  
Multiple accounts are stored using an **array of objects**, allowing the system to handle several users during program execution.

All operations such as deposit, withdrawal, and balance inquiry are performed using **member functions**, ensuring a clear separation between data and behavior.
